HOW TO RUN:
    1. start server first and the server will show "Server Running"
    2. start client and then enter the command like "listallfiles", "download file1.txt"
        or "exit"
    3. enter "exit" to quit from the client but will not quit from server


NOTE:
    1. if enter the command incorrectly, you will get the response from the server
        "please enter the correct command"
    2. when download the files, if the file not exist in the list of "listallfiles",
        you will get the response from the server "file does not exist"
